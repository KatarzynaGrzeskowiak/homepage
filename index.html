Tydzień 2

HTML

HTML-HyperText Markup Language. Odpowiada za właściwe opakowanie treści. Składa się ze znaczników, tagów, elementów.

Budowa znaczników: <znacznik atrybut="wartość" > treść </znacznik >

Semantyczny = mający znaczenie

HTML to szkielet strony

<main> główna cześć dokumentu

<section> sekcje dokumentu

<article> samodzielne treści

<header> nagłówek dokumentu/sekcji/arytkułu

<footer> stopka dokumentu/sekcji/arytkułu

<h1-h6> tekstowe nagłówki tytułowe dokumentu/sekcji/arytkułu

<p> paragraf/akapit

Walidator kodu html: W3 NU Validator

CSS

CSS - Cascading Style Sheets

Budowa reduły CSS selektor{właściwość:wartość;}

CSS możemy dodać w head w znaczniku <style></style> bądż podłączając plik w <link href="style.css" rel="stylesheet" type="text/css">
